# Changelog — 2026-03-29-RR

## Session Summary
Redesigned the waypoint elevation chart markers and added mobile swipe support for info tabs.

## Changes

### Elevation Chart Marker Redesign
- **File modified**: `components/routes/elevation-profile.tsx`
- All floating waypoint markers (S, 1, 2, ..., F) now use a unified dark green colour (`#267347`) matching the graph line — previously S was green, F was red, and numbered waypoints were blue
- Drop lines changed from coloured to soft grey (`#94A3B8`, 50% opacity) — subtle and non-distracting
- Drop lines now stop at the graph line intersection instead of extending to the bottom of the chart
- Intersection dots enlarged slightly (r=1.2) and coloured green to match the graph line

### Mobile Tab Swiping
- **File modified**: `components/routes/route-detail-drawer.tsx`
- Added touch swipe gesture support on the tab content area (Elevation, Waypoints, Hazards, Warnings, Variants, Weather)
- Swipe left/right to navigate between tabs on mobile
- Requires >50px horizontal movement and dominant horizontal direction to trigger
- Works alongside existing click-to-select behaviour
