## 13. Trust Score System Documentation

**Status:** Trust score system is fully implemented and working.

**How it works:**
- Each user starts with a base score of 30
- Score increases with:
  - Account age (+1 per month, max +15)
  - Social login (+10)
  - Bookings made/hosted (+3 each, max +15)
  - Reviews given (+2 each, max +10)
  - Accurate reports (+5 each, max +10)
- Score decreases with:
  - False reports (-10 each, max -20)
  - Warnings received (-10 each, max -30)

**Trust Levels:**
- 90+: Moderator
- 75+: Verified (internal status, not displayed to users)
- 50+: Trusted
- 25+: Standard
- <25: New

**Where it's used:**
- Admin inspection dashboard shows trust score
- Moderation system uses trust to prioritize reviews
- Report credibility is weighted by reporter's trust score

**Files:**
- `supabase/migrations/050_moderation_system.sql` - Database schema and functions
- `app/api/moderation/trust/route.ts` - API endpoint
- `components/admin/inspect-dashboard.tsx` - Admin display

## 14. Implemented Proper Image Cropping with react-cropper

**Problem:** No way to crop images before upload - they were auto-scaled by CSS which didn't give users control.

**Solution:** Installed `react-cropper` and `cropperjs` and created a reusable `ImageCropper` component.

**Features:**
- **Zoom control** - Slider to zoom in/out
- **Rotation** - Rotate left/right by 90°
- **Flip** - Horizontal and vertical flip
- **Aspect ratio** - Locked to appropriate ratio (1:1 for avatars, 4:3 for horse photos)
- **Preview** - See exactly what will be saved
- **Reset** - One-click reset to original

**Usage:**
- Profile pictures: Opens cropper after file selection, crops to square (1:1)
- Horse photos: Opens cropper, crops to 4:3 ratio
