## 1. New Reusable MobilePanelToggle Component

### Overview
Created a new reusable component for the Map/Options toggle button that provides consistent styling and behavior across all route tabs.

### File Created: `components/routes/mobile-panel-toggle.tsx`

### Design (OS Maps Style)
```
┌──────────────────┐
│   Map     🗺️    │  ← Small, discreet pill button
└──────────────────┘
```

### Button Specifications
- **Size**: Small pill shape (`h-9`, `px-5`)
- **Color**: Dark gray (`bg-gray-900`) with white text
- **Font**: 14px medium weight
- **Shadow**: Subtle elevation shadow
- **Icons**: Map icon for "Map" mode, SlidersHorizontal for "Options" mode
- **Animation**: Scale down on press (`active:scale-95`)
- **Fade-in**: Smooth opacity/translate animation when appearing

### Props
```tsx
interface MobilePanelToggleProps {
  mode: "map" | "options";      // Button type
  onClick: () => void;           // Click handler
  scrollContainerRef?: RefObject<HTMLElement>;  // Optional scroll detection
  alwaysVisible?: boolean;       // Skip scroll detection
}
```

## 2. Smooth Panel Slide Animations

### Overview
All mobile panels now use smooth slide animations when transitioning between map and details views.

### Animation Details
- **Duration**: 300ms
- **Easing**: `ease-out`
- **Panel slides down** when Map button clicked (reveals map underneath)
- **Panel slides up** when Options button clicked (covers map)

### CSS Implementation
```tsx
<div className={cn(
  "transition-transform duration-300 ease-out",
  panelOpen ? "translate-y-0" : "translate-y-full"
)}>
```
