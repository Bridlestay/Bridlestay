## 2. Quick Bookmark Button on Route Cards

### Overview
Added a bookmark icon to route cards for quick save/unsave functionality.

### Changes
- Bookmark icon in top-left corner of route thumbnail (Find Routes panel)
- Click to toggle saved/unsaved state
- Filled bookmark = saved, outline = not saved
- Toast notification confirms action

### Technical Implementation

**Popup Changes (`routes-map-mapbox.tsx`):**
- Disabled default Mapbox close button (`closeButton: false`)
- Added custom close button with white circular background
- Close button positioned absolute top-right over thumbnail
- Uses `onclick="this.closest('.mapboxgl-popup').remove()"` for removal
- "View details" button now removes popup before dispatching event

**Bookmark System (`find-routes-panel.tsx`):**
- Added state: `savedRouteIds` (Set of route IDs user has saved)
- Fetches user's favorites on component mount via `/api/routes/favorites`
- `toggleSaveRoute()` function handles save/unsave with optimistic UI
- Calls `POST /api/routes/[id]/favorite` to save
- Calls `DELETE /api/routes/[id]/favorite` to unsave
- Toast notifications for feedback
- Bookmark icon fills green when saved

### Files Modified
- `components/routes/routes-map-mapbox.tsx` - Popup styling improvements
- `components/routes/find-routes-panel.tsx` - Bookmark button on cards
