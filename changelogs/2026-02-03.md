## 5. UK-Only Route Creation

### Overview
Restricted route creation to UK & Ireland bounds only.

### Changes
- Added UK bounds constant: N 60.86°, S 49.86°, E 1.77°, W -8.65°
- Added `isWithinUK()` validation function
- When user clicks outside UK during route creation, shows toast error
- Message: "Routes can only be created within the UK & Ireland"

### Technical Implementation
```typescript
const UK_BOUNDS = {
  north: 60.86,  // Shetland
  south: 49.86,  // Channel Islands
  east: 1.77,    // Lowestoft
  west: -8.65,   // Western Ireland/Scotland
};

function isWithinUK(lng: number, lat: number): boolean {
  return lat >= UK_BOUNDS.south && 
         lat <= UK_BOUNDS.north && 
         lng >= UK_BOUNDS.west && 
         lng <= UK_BOUNDS.east;
}
```
